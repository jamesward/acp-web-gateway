# Demo

1. In an empty dir `megaclaw` have Kiro open
1. Run `acp2web --remote --agent-command \"kiro-cli acp\"`
1. Open the browser indicated in the output
1. Verify it works with a simple prompt like `what is in this dir`
1. Add the MCP server to Kiro
1. In Kiro, enter the prompt `add an ACP task to create an enteprise-grade OpenClaw (https://github.com/openclaw/openclaw)`
1. Show output in browser
